# Ctrl+S of Shame 😈

Ever saved broken code and felt nothing?

Not anymore.

**Ctrl+S of Shame** plays a dramatic scream whenever you:
- Save a file that contains errors
- Start debugging with errors

Because actions should have consequences.

---

## 🚀 Features

- 🔊 Plays a scream when saving files with errors
- 🐞 Triggers on debug start if errors exist
- ⏳ Cooldown protection (no spam)
- 🎭 Chaos energy built-in

---

## 📦 Installation

### From VS Code Marketplace

1. Open VS Code
2. Go to Extensions (`Ctrl+Shift+X`)
3. Search: **Ctrl+S of Shame**
4. Click Install

### From VSIX (Manual)

```bash
code --install-extension ctrl-s-of-shame-0.0.1.vsix
