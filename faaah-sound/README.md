# Ctrl+S of Shame

Every save is a gamble.

Ctrl+S of Shame plays a dramatic audio cue whenever you save code that contains errors.

## Features

- 🔊 Plays a scream when saving files with errors
- 🔁 Cooldown protection (no spam)
- 🎲 Random scream variations
- 🐞 Triggers on debug start with errors

## How It Works

When you:
- Save a file with errors
- Start debugging with errors

You are immediately reminded.

## Installation

Install from the Visual Studio Code Marketplace and enable it.

## Requirements

No additional setup required.
